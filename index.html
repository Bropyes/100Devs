What is CoinResolve?

CoinResolve is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making CoinResolve wallets a safer choice compared to desktop or browser extension wallets.

How does it work?

CoinResolve connects web applications to supported mobile wallets CoinResolve session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).

Frequently asked questions

→ How do i install CoinResolve

CoinResolve is not an app, but a protocol supported by many different decentralised applications and wallets. Install any of mobile wallets supporting CoinResolve protocol. CoinResolve wallets are available for Android and iPhone.

→ Is there a token?

There is no token. CoinResolve protocol does not run on a blockchain and there are no fees.

→ How can i help CoinResolve project?
